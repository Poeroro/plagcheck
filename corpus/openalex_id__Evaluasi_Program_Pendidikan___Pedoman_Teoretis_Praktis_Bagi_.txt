Evaluasi Program Pendidikan : Pedoman Teoretis Praktis Bagi Mahasiswa dan Praktisi pendidikan

Authors: Suharsimi Arikunto, Cepi Safruddin Abdul Jabar
Year: 2014

Setiap kegiatan manajemen akan dikatakan sempuma jika dalam prosesnya dilaksanakan suatu evaluasi, tidak terkecuali dalam manajemen pendidikan. Program pendidikan sebagai penjabaran dari perencanaan pendidikan harus dievaluasi dengan seksama, menggunakan strategi yang tepat sehingga hasilnya dapat dipertanggungjawabkan.
Evaluasi terhadap program pendidikan dimaksudkan untuk mengetahui tingkat keberhasilan atau kegagalan suatu program pendidikan. Hasil evaluasi tersebut dapat dijadikan informasi sebagai masukan untuk menentukan tindak lanjut dari program yang sedang atau telah dilaksanakan.
Melihat pentingnya evaluasi program pendidikan maka dalam buku ini disajikan berbagai hal penting berkenaan dengan evaluasi program pendidikan. Hal-hal yang dibahas antara lain, konsep dasar evaluasi program, model dan rancangan evaluasi program, perencanaan dan pelaksanaan evaluasi program, analisis hasil evaluasi, cara menyusun laporan evaluasi, dan tata tulis laporan evaluasi.
Pada edisi kedua ini dilakukan penyempurnaan, yakni menambah materi tentang program dengan komponen dan kriterianya, tujuan evaluasi program, pengambilan keputusan dan rekomendasi, serta analisis data dengan model SPSS.
